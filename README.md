# japanvel

## サイト概要
その観光地に初めて行くときにおすすめの観光箇所を見たい人

### サイトテーマ
初めての場所を初めてじゃない様に。

### テーマを選んだ理由
自分も旅行が好きで定番のところや定番ではないところなどに行きたい為

### ターゲットユーザ。
旅行好きな方。
観光でいつも通りの観光が嫌な方

### 主な利用シーン
初めて行く都道府県にでおすすめのところに行きたい時
